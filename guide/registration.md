# 注册账号

## 注册方式

Claude360 支持以下两种方式注册：

### GitHub 注册

1. 访问 [Claude360 官网](https://claude360.xyz)
2. 点击「GitHub 登录」
3. 授权 GitHub 账号
4. 完成注册

### Linux.do 注册

1. 访问 [Claude360 官网](https://claude360.xyz)
2. 点击「Linux.do 登录」
3. 授权 Linux.do 账号
4. 完成注册

::: tip 提示
两种方式注册后的账号功能完全一致，选择你常用的平台即可。
:::

## 注册后

注册成功后，您可以：

1. 创建专属 Key
2. 充值账户余额
3. 开始使用 AI 编程工具
